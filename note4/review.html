原型

原型 就是 函数的prototype属性所引用的对象。

原型的本质 就是 一个对象。

在声明函数时，JS解析器就会创建一个空对象，并将其赋值给函数的prototype属性。当通过该函数去实例化对象时，所创建出来的所有对象，都可以直接访问其原型对象（函数的prototype属性所指向的对象）的所有成员

获取原型

1 通过函数 .prototype 2 通过对象 .__proto__

原型-函数-实例关系

原型 默认有 constructor 指向 函数；负责将自己的成员 继承给 实例 father角色

函数 默认有 prototype 指向 原型；负责创建实例 mother 角色

对象 具有 __proto__ 指向 原型； son 角色

function extend( target ){
    if( !target ){
        return;
    }

    var args = arguments,
        i = 1,
        l = args.length;

    var k, obj;

    for( ; i < l; i++ ){
        obj = args[ i ];
        for( k in obj ){
            target[ k ] = obj[ k ];
        }
    }

    return target;
}
